# 语音备忘录 - 使用说明

## 📱 项目简介

这是一个响应式的语音备忘录网页应用，专为移动端语音输入和PC端管理设计。

**核心特点：**
- 📱 手机端：极简界面，专注语音输入
- 💻 PC端：完整管理功能，查看、搜索、导出
- 🎤 Web Speech API：原生浏览器语音识别
- 💾 本地存储：无需服务器，数据保存在浏览器

---

## 🎯 功能特性

### 移动端功能
1. **按住说话** - 按住圆形按钮开始录音，松开自动识别
2. **实时显示** - 识别结果实时显示在文本框
3. **快速保存** - 点击"保存备忘录"按钮即可保存
4. **备忘录计数** - 底部显示已保存的备忘录数量

### PC端功能
1. **统计面板** - 显示总备忘录数、今日新增、总字数
2. **搜索功能** - 实时搜索备忘录内容
3. **卡片展示** - 网格布局展示所有备忘录
4. **复制/删除** - 每条备忘录支持复制和删除
5. **导出功能** - 导出所有备忘录为JSON文件
6. **清空功能** - 一键清空所有备忘录

## 🚀 使用方法

### 1. 打开方式
直接用浏览器打开 `voice-memo.html` 文件即可。

### 2. 移动端使用（手机/平板）

#### 首次使用
1. 打开网页后，浏览器会请求麦克风权限
2. 点击"允许"授予权限

#### 录音步骤
1. **按住**中间的圆形按钮
2. 开始说话（按钮会变红，显示"正在录音..."）
3. **松开**按钮，等待识别
4. 识别结果会显示在下方文本框
5. 点击"✅ 保存备忘录"保存
6. 或点击"❌ 取消"重新录音

#### 注意事项
- 每次录音建议在10秒内完成
- 说话要清晰，避免环境噪音
- 识别需要网络连接

### 3. PC端使用（电脑）

#### 查看备忘录
- 打开网页自动显示所有备忘录
- 点击卡片可展开/收起完整内容

#### 搜索备忘录
- 在顶部搜索框输入关键词
- 实时过滤显示匹配的备忘录

#### 管理备忘录
- **复制**：点击📋图标复制内容到剪贴板
- **删除**：点击🗑️图标删除单条备忘录
- **导出**：点击"📥 导出"按钮导出JSON文件
- **清空**：点击"🗑️ 清空"按钮删除所有备忘录

## ⚠️ 注意事项

### 1. HTTPS要求
- 语音识别功能**必须**在HTTPS环境下使用
- 本地测试可以使用 `localhost` 或 `127.0.0.1`
- 部署到服务器需要配置SSL证书

### 2. 麦克风权限
- 首次使用会弹出权限请求
- 如果拒绝，需要在浏览器设置中重新授权
- Chrome: 设置 → 隐私和安全 → 网站设置 → 麦克风

### 3. 网络依赖
- 语音识别使用云端服务，需要网络连接
- 识别速度取决于网络质量
- 离线状态无法使用语音功能

### 4. 数据存储
- 数据保存在浏览器LocalStorage
- 清除浏览器数据会丢失所有备忘录
- 建议定期使用"导出"功能备份

### 5. 隐私安全
- 所有数据仅存储在本地浏览器
- 不会上传到任何服务器
- 语音识别由浏览器调用系统API

## 🐛 常见问题

### Q1: 点击按钮没有反应？
**A**: 
- 检查是否授予了麦克风权限
- 确认浏览器支持Web Speech API
- 尝试刷新页面重新授权

### Q2: 识别不准确怎么办？
**A**:
- 说话要清晰，语速适中
- 避免环境噪音干扰
- 尽量使用标准普通话
- 每次录音控制在10秒内

### Q3: 为什么识别一直失败？
**A**:
- 检查网络连接是否正常
- 确认麦克风硬件工作正常
- 尝试在其他浏览器中测试
- 查看浏览器控制台错误信息

### Q4: 数据会丢失吗？
**A**:
- 正常使用不会丢失
- 清除浏览器数据会丢失
- 建议定期导出备份
- 可以在PC端查看所有数据

### Q5: 可以在微信中使用吗？
**A**:
- 微信内置浏览器支持有限
- 建议使用系统浏览器打开
- iOS推荐Safari，Android推荐Chrome
